Dr. Awkward Palindrome - A Coding Project

A JavaScript coding exercise using objects, RegEx, loops, functions, and conditional statements. This program takes an input word or phrase and checks whether it qualifies as a palindrome. For example, Dr. Awkward is a palindrome.

Instructions

Type in a word or phrase and click the button to see if it is indeed a palindrome.
